# 💡 소개

- 서울시 공공데이터를 활용한 실시 인구 혼잡도 및 정보 제공 웹사이트
- https://lovelyseoul.vercel.app/

# ⭐️ 핵심 기능
- 인구 혼잡도 실시간 정보 및 가장 한산한 시간
- 문화 행사 정보 
- 현재 가장 인구가 많은 장소 추천
- 장소 북마크 기능
- 장소 기준 근처 문화 행사 정보 
- 장소명 검색 기능
